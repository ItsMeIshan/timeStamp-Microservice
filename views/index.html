TimeStamp Microservice

On /api/:date route provide the date as a parameter and you'll get an object {unix: timestamp, utc: timestamp}

Note: Date should be in format of YYYY-MM-DD

For Example: if date is 2021-09-26 then api route will be

[url]/api/2021-09-26

A request to /api/1451001600000 will return { unix: 1451001600000, utc: "Fri, 25 Dec 2015 00:00:00 GMT" }
